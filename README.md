# VoxAudit：智能催收语音质检与合规打分系统

VoxAudit 是一款专为金融零售信贷与集中式催收中心打造的 **AI 自动化质检解决方案**。系统旨在解决传统人工听录效率低、打分标准主观偏差大、以及合规风险（如过激言辞、隐私泄露）难以管控的业务痛点。

本项目依托 ASR（自动语音识别）与底层大语言模型（LLM），将非结构化的海量录音转化为可视化的双向对话，并基于可灵活配置的规则引擎，实现全量催收录音的**“100% 自动化听录、语义理解、智能打分与合规拦截”**。

## ✨ 核心特性 (Key Features)

* 🎙️ **高并发批量转写与双轨分离**
  * 支持百兆级别大文件（MP3/WAV/AMR）的分片与断点续传。
  * 精准声纹分离，自动切分「催收坐席」与「逾期客户」对话，携带精确时间戳。
* 🧠 **基于 LLM 的意图识别与智能打分**
  * 摒弃传统的关键词硬匹配，利用大模型强大的上下文理解能力，精准判断坐席是否完成规定动作（如：核实身份、告知逾期严重性）。
* 🛡️ **合规红线预警 (Compliance Guard)**
  * 内置零容忍风险词库与语义拦截模型，自动高亮并标记辱骂威胁、虚假承诺等违规话术，支持生成“红牌”复核工单。
* ⚙️ **租户隔离的动态规则引擎**
  * 支持各业务中心独立配置加扣分权重、一票否决项，提供规则版本控制与沙盒试运行能力。
* 📊 **多维度数据洞察**
  * 自动生成单据质检明细与聚合数据大盘，支持一键导出绩效核算报表。

## 🛠️ 技术栈 (Tech Stack)

* **后端架构**: Python (FastAPI / Flask) 
* **基础设施**: Docker / Docker Compose 容器化部署
* **数据存储**: MySQL (业务数据) + MinIO / OSS (语音文件存储) + Redis (异步任务队列)
